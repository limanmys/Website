# Liman Web Page

Liman MYS web sayfası

## Kullanım

1. Repo'yu klonlayın

```bash
git clone https://github.com/limanmys/website
```

2. Klonlanan repoya girin

```bash
cd website
```

3. Gerekli bağımlılıkları yükleyin

```bash
pnpm install
```

4. Geliştirme sunucusunu çalıştırın

```bash
pnpm run dev
```

5. Projeyi derleyin

```bash
pnpm build
```
